## 🚀 **GETTING STARTED**

### **📋 Prerequisites:**
```bash
# Install Flutter SDK
flutter doctor

# Verify installation
flutter --version
```

### **⚡ Quick Start:**
```bash
# Navigate to Flutter game
cd flutter_game

# Get dependencies
flutter pub get

# Run on your platform of choice:

# Android
flutter run -d android

# iOS (macOS only)
flutter run -d ios

# Web
flutter run -d chrome

# Windows
flutter run -d windows

# macOS
flutter run -d macos

# Linux
flutter run -d linux
```

### **🏗️ Build for Production:**
```bash
# Android APK
flutter build apk --release

# iOS App Store
flutter build ios --release

# Web deployment
flutter build web --release

# Windows executable
flutter build windows --release

# macOS app
flutter build macos --release

# Linux app
flutter build linux --release
```

## 🚀 **HOW TO RUN YOUR MOBILE GAME**

### **📋 Quick Start:**
```bash
# Navigate to mobile game
cd mobile_game

# Install dependencies  
npm install

# For Android
npm run android

# For iOS (macOS only)
npm run ios
```

### **📱 Build for App Stores:**
```bash
# Android Release (Google Play Store)
npm run build:android

# iOS Release (Apple App Store)  
npm run build:ios
```
